Project Overview
This project is part of my public portfolio and showcases my work at the intersection of credit risk modeling and machine learning. The objective is to develop and evaluate models for estimating Expected Loss (EL) in retail credit, using the formula:

EL=PD×LGD×EAD

The analysis is based on the Home Credit Default Risk dataset from Kaggle.

Scope and Goals

The project focuses on:

Estimating Probability of Default (PD) using two modeling approaches:

Logistic Regression, the traditional market standard in credit risk

XGBoost, a modern gradient‑boosting benchmark known for strong predictive performance

Constructing a full Expected Loss framework, incorporating PD, LGD, and EAD assumptions aligned with credit‑risk best practices.

Comparing model performance using both statistical metrics and business‑oriented evaluation criteria.

Producing an executive‑level report summarizing insights, model behavior, and implications for credit‑risk decision‑making.
